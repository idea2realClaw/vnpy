## 二、最接近的反向工程公式

### 第 1 步：把每个指标转成滚动百分位
对第 i 个指标的时间序列 `m_i`，在第 t 日计算其在**过去 T 个交易日（含当日）滚动窗口**内的百分位排名：

```
r_i(t) = ( 1 / T ) · Σ_{k=1..T} 1{ m_i(t−k+1) ≤ m_i(t) }      T ≈ 1260（≈5 年滚动）
```

> 说明：VTS 用 "rolling percentiles"，即每个指标相对自身历史窗口的百分位。用严格 `<` 还是 `<=` 对连续分布影响可忽略，这里取严格 `<`（与 daofund 的 percent_rank 一致）。T 取 5 年（1260 交易日）对应"需 5 年数据排百分位"。

### 第 2 步：方向校正（处理反向指标）
```
q_i(t) = r_i(t)            # 正常指标：指标越高 = 波动率越高 = 百分位越高
q_i(t) = 1 − r_i(t)        # 反向指标（约 2 个，如 VRP 类、VIX Oscillator）：极端低波动时不归零
```

### 第 3 步：加权汇总成 0–100
```
Barometer(t) = 100 · Σ_i w_i · q_i(t)  /  Σ_i w_i
```
未知权重时取等权 `w_i = 1`。

## 五、结论与置信度

- **方法论公式（第 1–3 步）是 VTS 亲述的，置信度最高**——"13 个波动率指标 → 各自滚动百分位 → 加权汇总成 0–100"。可直接用于复现其资产轮动逻辑。
- **具体哪 13 个指标是推断的**（置信中）：VTS 故意保密，社区克隆版（TradingView VB）只用 3 个（VIX/VVIX/期限结构）且用 z-score 而非百分位，与原版不同，仅供参考。
- **权重假设等权**（置信低）：原版权重未公开，等权是最公平的默认。
- **最重要的事实**：原版用滚动百分位 + 含 2 个反向指标，所以**永远不会到 0/100**，且 2017 极端低波动时也只到 13.82%——这恰好解释了为什么上一轮我们单纯用"VIX 10 年百分位"在 2017 会过度空仓（缺反向指标的平衡）。

> 实战建议：把本公式接回 `run_vts_backtest.py` 的 Barometer，替换原先的 p1_only / composite，用校验锚点对齐后驱动 Defensive Rotation / Strategic Tail Risk 轮动，会比之前更接近 VTS 真实行为。

## 六、实测校验结果（2026-07-17 运行；依次补抓 ^VVIX / ^VIX9D 真实数据后重算）

数据：SQLite 中 VIX / VXV(=^VIX3M) / **VVIX(=^VVIX，真实 2007+)** / **VIX9D(=^VIX9D，真实 2011+)** / SPY，均延伸到 2006；等权 **16 个百分位指标（含 3 个反向：VIX_osc / VRP_osc / VVIX_VIX_ratio）**，T=1260（5 年）滚动窗口，warmup 前 1259 根置 NaN。VIX9D 因 2011 才起，不对索引做交集截断——缺失期(2011 前)与自身 5 年 warmup 置中性 0.5，保留 2007-2011 历史上下文。

| 校验项 | VTS 公布 | 代理基线(15) | +VVIX(15) | +VVIX9D(16) | 评价 |
|---|---|---|---|---|---|
| 全样本均值 | 46.62% | 42.91% | 42.88% | **43.26%** | ✅ 单调逼近（差 3.4） |

> 代理基线 = 仅用 VIX/VXV/SPY 可计算指标（VVIX、VIX9D 尚未补真实数据，其分量缺失）。可见每补一个真实短/中/长窗口 vol 数据源，所有锚点指标**单调**更接近 VTS 公布值。

**锚点日实测读数**（关键，加 VVIX9D 后）：
- 2017-07-21 → **13.69**（VTS 13.82，**误差仅 0.13，几乎命中**）
- 2020-03-11 → **86.31**；2020-03-12 → **85.01**（VTS 峰值 90.95）
- 2020-03-16 → **86.34**（全局新最高，新冠崩盘期）
- 2025-04-10 → **85.24**
- 全局最高锁定 **2020-03-16（新冠）**，与 VTS 公布峰值日(2020-03-12)一致；2025 关税(85.24)退居次高。

**VVIX + VIX9D 的改进**：
- **VVIX** 打破"顶部封顶 ~84"——峰值 83.69→85.54、峰值时点从 2025 关税纠正回 2020 新冠（vol of vol 危机中创历史新高拉满百分位）。
- **VIX9D** 进一步全面收紧：均值 +0.38（唯一抬升中枢的一步，因短窗口 vol 在平静期也常处中高位）、峰值 85.54→86.34、2017 低点读数 14.49→13.69（几乎命中 13.82）、<20% 与 >80% 尾部双双逼近公布值。VIX9D 在危机中尖峰最陡、百分位逼近 1.0，直接补强顶部极端。

**剩余差距**：① 均值仍低 3.4（43.3 vs 46.6）；② >80% 尾部 2.0% vs 5.6% 仍偏紧。进一步逼近的可行路径：① 构造 **VIX 期货期限结构斜率（近月−次月）** 替换当前 VIX−VIX3M 代理（VIX 期货本地仍缺）；② 保持等权还原哲学下，可选对汇总值做单调拉伸（min-max/分位重标定到 VTS 公布分布）——但属拟合非还原，本版仍保持等权原样、如实呈现差距。

**结论**：依次补抓 VVIX、VIX9D 真实数据后，公式在均值(43.3 vs 46.6)、峰值(86.3 vs 90.9 @2020)、2017 低点读数(13.7 vs 13.8)、Volpocalypse 前后段(33.6/51.4 vs 39.9/53.3)上**单调逼近** VTS 公布锚点，是当前**可计算、无未来函数、最接近原版**的重建。图表：`vts_barometer_chart.html`。
