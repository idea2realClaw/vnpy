## 二、最接近的反向工程公式

### 第 1 步：把每个指标转成滚动百分位
对第 i 个指标的时间序列 `m_i`，在第 t 日计算其在**过去 T 个交易日（含当日）滚动窗口**内的百分位排名：

```
r_i(t) = ( 1 / T ) · Σ_{k=1..T} 1{ m_i(t−k+1) ≤ m_i(t) }      T ≈ 1260（≈5 年滚动）
```

> 说明：VTS 用 "rolling percentiles"，即每个指标相对自身历史窗口的百分位。用严格 `<` 还是 `<=` 对连续分布影响可忽略，这里取严格 `<`（与 daofund 的 percent_rank 一致）。T 取 5 年（1260 交易日）对应"需 5 年数据排百分位"。

### 第 2 步：方向校正（处理反向指标）
```
q_i(t) = r_i(t)            # 正常指标：指标越高 = 波动率越高 = 百分位越高
q_i(t) = 1 − r_i(t)        # 反向指标（约 2 个，如 VRP 类、VIX Oscillator）：极端低波动时不归零
```

### 第 3 步：加权汇总成 0–100
```
Barometer(t) = 100 · Σ_i w_i · q_i(t)  /  Σ_i w_i
```
未知权重时取等权 `w_i = 1`。

## 五、结论与置信度

- **方法论公式（第 1–3 步）是 VTS 亲述的，置信度最高**——"13 个波动率指标 → 各自滚动百分位 → 加权汇总成 0–100"。可直接用于复现其资产轮动逻辑。
- **具体哪 13 个指标是推断的**（置信中）：VTS 故意保密，社区克隆版（TradingView VB）只用 3 个（VIX/VVIX/期限结构）且用 z-score 而非百分位，与原版不同，仅供参考。
- **权重假设等权**（置信低）：原版权重未公开，等权是最公平的默认。
- **最重要的事实**：原版用滚动百分位 + 含 2 个反向指标，所以**永远不会到 0/100**，且 2017 极端低波动时也只到 13.82%——这恰好解释了为什么上一轮我们单纯用"VIX 10 年百分位"在 2017 会过度空仓（缺反向指标的平衡）。

> 实战建议：把本公式接回 `run_vts_backtest.py` 的 Barometer，替换原先的 p1_only / composite，用校验锚点对齐后驱动 Defensive Rotation / Strategic Tail Risk 轮动，会比之前更接近 VTS 真实行为。

## 六、实测校验结果（2026-07-17 运行；2026-07-17 补抓 ^VVIX 真实数据后重算）

数据：SQLite 中 VIX / VXV(=^VIX3M) / **VVIX(=^VVIX，真实数据 2007+)** / SPY，均延伸到 2006；等权 **15 个百分位指标（含 3 个反向：VIX_osc / VRP_osc / VVIX_VIX_ratio）**，T=1260（5 年）滚动窗口，warmup 前 1259 根置 NaN。

| 校验项 | VTS 公布 | 重建值(加VVIX) | 之前(代理) | 评价 |
|---|---|---|---|---|
| 全样本均值 | 46.62% | **42.88%** | 42.91% | ✅ 接近（差 3.7） |
| 历史最低 | 13.82% @ 2017-07-21 | **9.79% @ 2017-01-24** | 8.08% @ 2017-10-05 | ✅ 同期(2017 低波动年)，值偏低 |
| 历史最高 | 90.95% @ 2020-03-12 | **85.54% @ 2020-03-16** | 83.69% @ 2025-04-10 | ✅ 峰值回到 2020 新冠(方向正确)，顶部 83.7→85.5 封顶缓解 |
| <20% 占比 | 5.2% | 8.9% | 9.3% | ⚠️ 偏低方向仍偏多 |
| >80% 占比 | 5.6% | 1.7% | 0.9% | ⚠️ 顶部仍偏紧(但翻倍) |
| Volpocalypse 前 7 年均值 | 39.94% | **33.11%** | 34.81% | ✅ 接近 |
| Volpocalypse 后 7 年均值 | 53.31% | **51.03%** | 50.27% | ✅ 接近 |

**锚点日实测读数**（关键，加 VVIX 后）：
- 2017-07-21 → **14.49**（VTS 13.82，差 0.67，非常准）
- 2020-03-11 → **85.52**；2020-03-12 → **84.14**（VTS 峰值 90.95）
- 2020-03-16 → **85.54**（全局新最高，新冠崩盘期）
- 2025-04-10 → **84.42**
- 全局最高锁定在 **2020-03-16（新冠）**，而非上一版的 2025-04-10（关税）—— **方向已纠正，与 VTS 公布峰值日(2020-03-12)一致**。

**VVIX 带来的改进**：打破此前"顶部封顶 ~84"的系统性差距——峰值从 83.69% 抬升到 85.54%、且峰值时点正确回到 2020 新冠危机（之前被 2025 关税误占）。根因：VVIX 在危机中创历史新高(~190)，其百分位逼近 1.0，作为正常指标强力拉动顶部；VVIX/VIX 比作为反向指标在危机时比值骤降、取反后同样推高顶部。均值与前后段基本持平，说明 **VVIX 主要修正尾部极端、而非移动中枢**。

**剩余差距**：① 均值仍低 3.7（42.9 vs 46.6）；② >80% 尾部仍偏紧（1.7% vs 5.6%）。进一步逼近的可行路径：① 补抓 **^VIX9D**（短窗口 vol 在危机中尖峰最陡，直接补强顶部——成本极低，强烈建议下一步）；② 构造 VIX 期货期限结构斜率（近月−次月）替换当前 VIX−VIX3M 代理；③ 保持等权还原哲学下，可选对汇总值做单调拉伸（min-max/分位重标定到 VTS 公布分布）——但属拟合非还原，本版仍保持等权原样、如实呈现差距。

**结论**：加入真实 VVIX 后，公式在峰值时点(2020 新冠)、峰值高度(85.5 vs 90.9)、2017 低点读数(14.5 vs 13.8)、Volpocalypse 前后段均与 VTS 公布锚点一致，是当前**可计算、无未来函数、最接近原版**的重建。图表：`vts_barometer_chart.html`。
